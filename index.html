Name Tag

Create and manage name tags for broadcast.

  • Enter name and title
  • Choose position (left, center, right)
  • Customize color and gradient
  • Save up to 5 presets for quick access
  • Adjust rounded corners
Open Name Tag

Bitfocus Companion Integration

Both tools can be controlled via Companion using HTTP requests:

Name Tag:

  • Load preset: http://[IP]/nametag.html?tag=1 (1-5)
  • Change name: http://[IP]/nametag.html?name=John%20Doe
  • Change title: http://[IP]/nametag.html?title=Producer
  • Change position: http://[IP]/nametag.html?position=left (left/center/right)

Color Background:

  • Change color directly by opening the page
  • Example: http://[IP]/color-background.html

Tips

  • Hover at the top of the page to show the control panel in both tools
  • All settings are automatically saved in the browser
  • Name tag presets can be named for easy identification
  • Gradient effects can be used in both tools
